# 軽貨物まるごとアプリ

軽貨物ドライバー向けの「点呼・業務簿・お金まわり」をひとつにまとめたスマホアプリです。
北上さん＋同僚ドライバーが、それぞれ自分のスマホで使います。

## 公開URL

**https://nylonman1109-prog.github.io/keikamotsu-app/**

このURLを開けばそのまま使えます。ホーム画面に追加すれば、普通のアプリのようにアイコンから起動できます（手順は下記）。

## 使い方（かんたん3ステップ）

1. 上のURLをスマホのブラウザ（Safari／Chrome）で開く
2. 「新規登録」からメールアドレスとパスワードを決めて登録
3. ログインしたら、下のタブ（ホーム／レシート／個数／グラフ／🚛運行記録）を使って記録していく

- 収入・経費は「ホーム」の「収入を記録」ボタンや「レシート」タブから
- 配達個数は「個数」タブから
- 点呼簿・業務簿・車両点検は「🚛運行記録」タブから
- 「今月の損益」は自動でホームに表示されます

## 同僚への配り方

同じURL（https://nylonman1109-prog.github.io/keikamotsu-app/）を送るだけです。
各自が自分のメールアドレスで新規登録すれば、データは自動的に本人だけのものとして分かれて保存されます（他人のデータは見えません）。

## まとめ役について

北上さんのアカウント（nylonman1109@gmail.com）は「まとめ役」として、ホームの「👥全員を見る」からメンバー全員の状況（今日の配達個数・損益など）をまとめて確認できます。他のメンバーは自分の分しか見えません。

## ホーム画面に追加する（アプリのように使う）

### iPhone（Safari）

1. Safariで公開URLを開く
2. 画面下の「共有」ボタン（四角から矢印が上に出ているアイコン）をタップ
3. メニューを下にスクロールして「ホーム画面に追加」をタップ
4. 名前はそのまま（「軽貨物まるごと」）で右上の「追加」をタップ
5. ホーム画面にアイコンが増える → タップすればブラウザのアドレスバー無しでアプリのように起動

### Android（Chrome）

1. Chromeで公開URLを開く
2. 右上の「︙」（縦三点メニュー）をタップ
3. 「アプリをインストール」または「ホーム画面に追加」をタップ
4. 表示された内容を確認して「インストール」（または「追加」）をタップ
5. ホーム画面／アプリ一覧にアイコンが増える

一度追加すれば、2回目以降はオフラインでも画面の外枠（ログイン画面など）はすぐ表示されます。ただし実際のデータ（売上・経費・配達個数など）はインターネットに繋がっていないと読み込み・保存ができません。

## 更新方法（北上さん向け）

1. `projects\軽貨物まるごとアプリ\index.html` を直す（機能追加・修正はここ）
2. 作業用リポジトリで `git add` → `git commit` → `git push`
3. 数分でGitHub Pagesに自動反映される（公開URLがそのまま更新される）

アイコンやPWAの設定を変えたい場合は、同じフォルダの `manifest.json`（アプリ名・色・アイコンの指定）と `icons/` フォルダ（アイコン画像）、`sw.js`（オフライン時の外枠キャッシュ）を編集してから同様にpushしてください。

## ファイル構成

```
projects/軽貨物まるごとアプリ/
├── index.html          … アプリ本体（Firebase接続済み）
├── manifest.json        … PWA設定（アプリ名・色・アイコン指定）
├── sw.js                 … Service Worker（外枠だけキャッシュ・データは常にオンライン）
├── icons/
│   ├── icon-192.png      … Android等向けアイコン
│   ├── icon-512.png      … 高解像度アイコン
│   └── apple-touch-icon.png … iOS向けアイコン(180x180)
├── firebase_rules.json   … Realtime Databaseのセキュリティルール（正本）
├── 企画書_軽貨物まるごとアプリ.md
├── モックアップ_強化版_v1.html   … 見た目確認用（現行版とは別に温存）
├── モックアップ_お金まわり.html   … 見た目確認用（旧）
└── handoffs/              … 開発の引き継ぎ記録（LATEST.mdが最新）
```

## 困ったときは

- ログインできない／データが表示されない → 電波（Wi-Fi/モバイル通信）が繋がっているか確認
- 表示が崩れる／古いままに見える → ホーム画面のアイコンから一度アプリを完全に閉じて開き直す
- それでも直らない → 北上さんに連絡
